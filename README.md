# LynxHR-Client-arm64
LynxHR Client for ARM64 Based Watches running Android

# Supported Devices
- Fossil Smart Watches (Gen 4 and Newer)
- Google Pixel Watch
- Samsung Galaxy Watch 4
- Samsung Galaxy Watch 5
